# Trabalho 1: Navegação e Interação do Robô NAO no CoppeliaSim

Este trabalho para a disciplina de **Robôs, Sensores e Aplicações** implementa uma simulação de controle para o robô NAO. O robô é posicionado em um ambiente de jardim que serve como arena, contendo plantas como obstáculos e bordas que funcionam como barreiras de segurança.

O desafio é programar o NAO para:

1.  Navegar pelo ambiente de forma autônoma, desviando dos obstáculos.
2.  Identificar e se aproximar de cubos coloridos.
3.  Ao chegar perto de um cubo, executar uma ação (levantar a mão) e interagir (falar a cor do cubo).
